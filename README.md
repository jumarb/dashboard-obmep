# 📊 Dashboard OBMEP

Análise interativa dos dados de medalhas da Olimpíada Brasileira de Matemática das Escolas Públicas (OBMEP), referente às edições 16ª a 20ª.

## 🔗 Acesse o dashboard

[Clique aqui para acessar](https://dashboard-obmep-kmhhiuhzrxvzkuvffwfafa.streamlit.app/)

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um portfólio de análise de dados, combinando dados reais da OBMEP com ferramentas de visualização interativa.

Os dados foram extraídos de documentos oficiais da OBMEP e organizados manualmente, cobrindo os três níveis de competição (1, 2 e 3) ao longo de cinco edições.

## 📊 O que o dashboard mostra

- Evolução do total de medalhas por edição
- Comparação entre rede pública e privada
- Distribuição por sexo
- Evolução do percentual público por nível
- Filtro interativo por nível de competição

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- Plotly
- Streamlit
